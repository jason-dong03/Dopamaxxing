export function pickRarityFromWeights(
    weights: Record<string, number>
): string {

    const total = Object.values(weights)
        .reduce((a, b) => a + b, 0)

    let roll = Math.random() * total

    for (const [rarity, weight] of
         Object.entries(weights)) {
        roll -= weight
        if (roll <= 0) return rarity
    }

    return 'Common'
}
PLAIN ENGLISH
This function picks a rarity using a weighted lottery. Give it a table of weights, get a rarity back.
Add up all the weights to find the total size of the lottery space. (Example: 78 + 16 + 4.5 + … = ~100)
Roll a random number anywhere from 0 to that total — this is the dart thrown at the lottery board.
Walk through each rarity one by one, from top to bottom of the list.
Subtract that rarity's weight from the roll — like crossing off lottery numbers.
The moment the roll hits zero or below — that rarity wins the lottery and gets returned.
If nothing hit (floating point rounding edge case) — fall back to Common as a safe default.

The Pity System: No More Infinite Bad Luck

Every pack you open without a Legendary+ increments a counter in the database column called pity_counter. When that counter reaches pity_threshold, the machine secretly swaps in a guaranteed Legendary+ ticket — then resets the counter to zero.

Why every gacha game has this

This pattern — guaranteed reward after N attempts — is used by nearly every gacha game ever made. Game designers call it a soft pity or mercy system. It exists because humans have a powerful emotional reaction to feeling cheated by pure randomness — the math makes the game feel fair even when you are unlucky.

Your Card's Condition: The Four Attributes

Every card that drops gets four random condition scores on a 1–10 scale. These scores are combined into a weighted grade that determines the card's actual sell price — just like PSA grading for physical Pokémon cards.

⊞

Centering

How centered the card image is within its border. A perfectly centered card looks more valuable — off-center prints look mass-produced and cheap.

◇

Corners

Sharpness of all four corners. This has the highest weight because corners show wear first — a bent corner is the most visible sign of damage on a real card.

▭
25% weight
Edges

Smoothness of the card edges. Nicks and chips reduce value — a pristine edge means the card was never roughly handled or stored loose in a pile.

◻
20% weight
Surface

Scratches, print lines, and marks on the card face. Surface damage is hardest to spot but kills a card's grade under a grader's loupe.
